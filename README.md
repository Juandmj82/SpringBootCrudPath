# 🚀 SpringBootCrudPath

Este repositorio está organizado por niveles para facilitar mi práctica y consulta de conceptos.

## 🛠️ Tecnologías Core

- **Java 17+**
- **Spring Boot 3.x**
- **Spring Data JPA**
- **MapStruct** (Mapping)
- **Lombok** (Boilerplate)
- **H2 Database** (Runtime Test)

---

## 📝 Cómo usar este repositorio

Cada carpeta es un proyecto de Maven independiente. Puedes abrir cualquier nivel directamente en tu IDE preferido (IntelliJ, VS Code, Eclipse) para estudiar su implementación.

> [!TIP]
> Si quieres practicar, usa el comando `git restore .` para descartar tus cambios locales y volver a la versión original en cualquier momento.
