# 🚀 SpringBootCrudPath

Bienvenido a **SpringBootCrudPath**, un repositorio secuencial diseñado para dominar el desarrollo de APIs REST con Spring Boot, desde los fundamentos hasta patrones de arquitectura industrial.

Este repositorio está organizado por niveles de complejidad creciente, sirviendo como una ruta de aprendizaje y manual de referencia técnico.

---

## 🛠️ Tecnologías Core

- **Java 17+**
- **Spring Boot 3.x**
- **Spring Data JPA**
- **MapStruct** (Mapping)
- **Lombok** (Boilerplate)
- **H2 Database** (Runtime Test)

---

## 📝 Cómo usar este repositorio

Cada carpeta es un proyecto de Maven independiente. Puedes abrir cualquier nivel directamente en tu IDE preferido (IntelliJ, VS Code, Eclipse) para estudiar su implementación.

> [!TIP]
> Si quieres practicar, usa el comando `git restore .` para descartar tus cambios locales y volver a la versión "Pro" original en cualquier momento.
